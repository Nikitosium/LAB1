**ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32-БІТНІ МК**
**Лабораторна робота 1.** Технологія програмування портів вводу-виводу загального призначення на основі прототипу операційної системи нескінченного циклу опитування.

**Обладнання:** STM32F411 Discovery, CubeMX, Keil.

**Мета:** Ознайомитися з принципами програмування GPIO STM32F411 через регістрові структури, реалізувати нескінченний цикл опитування кнопки та керування світлодіодами.

__________________________________________________________________________________________

**Завдання до лабораторної роботи (Варіант 4):** Реалізувати інверсію стану світлодіода **PD15** при кожному натисканні кнопки **PA0**.

Цей проєкт реалізує низькорівневу роботу з GPIO на мікроконтролері STM32 з використанням прямого доступу до регістрів (CMSIS). Програма відстежує фронт натискання кнопки та змінює стан світлодіода на протилежний.

### 👨🏼‍💻 Функціонал:
* **Button Debounce**: Реалізовано програмну затримку для уникнення впливу брязкоту контактів кнопки PA0 на логіку перемикання.
* **Edge Detection**: Алгоритм фіксує момент переходу кнопки зі стану "відпущено" в "натиснуто", що запобігає безконтрольному миготінню при затисканні кнопки.
* **Register-level control**: Конфігурація периферії здійснюється через безпосередній запис у регістри MODER, PUPDR та RCC без сторонніх драйверів.

### 📋 Ключові моменти реалізації:
1. **Clock Control**: Активація тактування шини AHB1 для портів GPIOA та GPIOD.
2. **Pin Configuration**: Налаштування 15-го піна порту D на вихід та 0-го піна порту A на вхід.
3. **Toggle Logic**: Використання побітового виключного "АБО" (XOR) для зміни стану конкретного біта в регістрі ODR.
